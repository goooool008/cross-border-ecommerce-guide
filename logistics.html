2025-2026 重大变化

  • 美国T86免税清关通道关闭
  • 欧盟取消150欧元以下小包免税
  • 小包直邮"免税红利"彻底终结
  • 海外仓+FBA成为主流选择

不同商品的最优方案

轻小件低客单价 (< $20)

测款用小包直邮，起量后转海外仓/FBA，成本降 20-35%

中大件高客单价 (> $50)

海运拼柜+海外仓最划算，比直邮省 40-60%

季节性 / 时尚品类

爆款提前备货海外仓，非爆款走专线

混合模式推荐

  1. 测款新品 → 小包直邮 / 专线空运（零风险验证）
  2. 起量爆款 → FBA（蹭Prime流量 + 平台权重）
  3. 多平台订单 → 第三方海外仓一件代发（库存复用）
  4. FBA库容紧张 → 海外仓中转分批补仓
  5. FBA退货 → 海外仓换标翻新，重新入仓
新手建议：测款用小包直邮（零风险验证）→ 起量后转FBA/海外仓（降本提效）。2026年不再是"选哪个最便宜"，而是根据商品属性、客单价、周转率动态组合。
